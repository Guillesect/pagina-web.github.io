Ejemplos de CSS

Propiedad: background-color

La propiedad background-color se usa para establecer el color de fondo de un elemento.

div {
    background-color: lightblue;
}

Resultado:

Este div tiene un fondo de color azul claro.

Propiedad: margin

La propiedad margin define el espacio alrededor de un elemento.

div {
    margin: 20px;
}

Resultado:

Este div tiene un margen de 20px.

Propiedad: font-family

La propiedad font-family permite definir la fuente de texto de un elemento.

p {
    font-family: 'Arial', sans-serif;
}

Resultado:

Este párrafo tiene la fuente Arial.

Propiedad: text-align

La propiedad text-align se utiliza para alinear el texto dentro de un elemento.

h1 {
    text-align: center;
}

Resultado:

Título centrado

Propiedad: font-size

La propiedad font-size ajusta el tamaño de la fuente de un elemento.

p {
    font-size: 40px;
}

Resultado:

Este párrafo tiene un tamaño de fuente de 40px.
