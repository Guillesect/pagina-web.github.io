Ejemplos de SVG

<circle> - Círculo

La etiqueta <circle> se utiliza para dibujar un círculo dentro de un elemento SVG.

<svg width="100" height="100">
  <circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red" />
</svg>

Resultado:

<rect> - Rectángulo

La etiqueta <rect> se utiliza para dibujar un rectángulo.

<svg width="200" height="200">
  <rect width="150" height="100" stroke="black" stroke-width="5" fill="green" />
</svg>

Resultado:

<line> - Línea

La etiqueta <line> se utiliza para dibujar una línea entre dos puntos.

<svg width="200" height="200">
  <line x1="0" y1="0" x2="200" y2="200" stroke="blue" stroke-width="4" />
</svg>

Resultado:

<text> - Texto

La etiqueta <text> se utiliza para agregar texto en un área SVG.

<svg width="200" height="200">
  <text x="50" y="50" font-family="Verdana" font-size="35" fill="blue">Hola</text>
</svg>

Resultado:

<polygon> - Polígono

La etiqueta <polygon> se utiliza para dibujar un polígono con una serie de puntos.

<svg width="200" height="200">
  <polygon points="50,150 150,150 100,50" stroke="black" stroke-width="3" fill="yellow" />
</svg>
