Ejemplo de etiquetas HTML

<h1> - Encabezado principal

La etiqueta <h1> se usa para el encabezado principal de una página web.

<h1>Mi primer encabezado</h1>

Resultado:

<p> - Escribir párrafos

La etiqueta <p> se usa para escribir párrafos en la página web.

<p>Mi primer párrafo</p>

Resultado:

<a> - Insertar enlaces

La etiqueta <a> se usa para poder insertar enlaces en una página web.

El destino del enlace se especifica en el atributo href. Los atributos se utilizan para proporcionar información adicional sobre elementos HTML.

<a href="https://ejemplo.com">Enlace de ejemplo</a>

Resultado:

<img> - Insertar imágenes

La etiqueta <img> se usa para insertar imágenes en una página web.

El archivo fuente (src), texto alternativo (alt), ancho y alto se proporcionan como atributos.

<img src="Miimagen.jpg" alt="primera imagen" width="100" height="140">

Resultado:

<body> - Cuerpo de la página

La etiqueta <body> es la parte visible del documento HTML de una página web.

<body><p>Mi primer párrafo</p></body>

Resultado:
